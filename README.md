Nosso trabalho consistiu em fazer uma loja com estoque de produtos atravéz de cadastro por ID com acrescimo e decrescimo de itens dependendo da adição no carrinho, 
cadastro de clientes, 
cadastro de vendedores, 
aplicação de desconto, 
adição de frete para os produtos que fossem sair para delivery e impressao do relatório de vendas.
